기능 구현 목록
---

##### 자료구조
- 게임에 사용될 3자리 배열 : Number class
- 볼 카운트 데이터        : Ballcount class

##### 컴퓨터 숫자생성
- 컴퓨터 난수 생성 : createNumber()

##### 게임 메세지 출력
- 게임 가이트 메세지 출력 : guideMessage()

##### 사용자 입력
- 사용자의 값 입력 : input()

##### 사용자 숫자 검사
- 1~9이외 값, 중복 검사확인 : userNumCheck()

##### 볼 카운트 검사
- 스트라이크, 볼인지 검사 : playGame()

##### 볼 카운트 메세지 출력
- X 스트라이크 X 볼 : printBallCount()
